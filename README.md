# Offset Polygon

Small, no dependency library for offsetting polygons. Heavily based on this [CodePen](https://codepen.io/HansMuller/pen/lDfzt) by Hans Muller.

## Usage

```csharp
PolygonOffsetHelper.OffsetPolygon(points, -10);
```
